### 列表指针的使用

```
"""
给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

"""
nums = [0,0,1,1,1,2,2,3,3,4]
a, b = 0, 1 #设置初始指针
while b < len(nums):
    if nums[a] == nums[b]:
        nums.pop(b)
    else:
        a, b = a + 1, b + 1  #每次指针向前移动一位
print(len(nums))
print(nums)
```
