# Python3 和 Python2的区别
| 区别    | Python2                                       | Python3           |
| ------- | --------------------------------------------- | ----------------- |
| 源码    | 不统一，有重复代码                            | 统一，无重复代码  |
| 除法    | 返回整型                                      | 返回浮点型        |
| input() | 与输入类型相同；raw_input()获取到的都是字符串 | 均返回字符串      |
| print   | print "输出内容"                              | print('输出内容') |
