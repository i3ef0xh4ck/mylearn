## 列表

+ 定义
  + lst = [1,"xxx",True]   每个元素用,分隔
  + 可变、可迭代、可修改的有序的数据类型
  + 实际存储的是内存地址，指向对应数据

+ 作用
  
  + 容器
+ 存储大量数据
  
  + 存储不同类型的数据
  
+ 列表的操作

  + 增：添加列表元素

    + append  末尾追加

      ```
      lst = ["Python学习手册","核心编程"]
      lst.append("流畅的Python")
      print(lst)
      ```

    + insert 插入（按索引）

      参数1：插入位置的下标（理解为插入后目标索引位置） 参数2：插入的内容

      ```
      lst = ["Python学习手册","核心编程"]
      lst.insert(1,"流畅的Python") 
      print(lst)
      ```
```
      
      注：尽量少使用，影响较大
      
    + extend 迭代增加
    
      参数为可迭代对象
    
```
      lst = ["Python学习手册","核心编程"]
      lst.extend("流畅的Python")

      ```
    
      字符串转换列表的三种方式
    
      ```

      print(a)

  + 删：删除列表中的元素
  
    实质是删除内存地址
  
    + remove 通过元素名进行删除
    
      ```
      lst = ["大圣","海","心理","一帆","靓仔","石头"]
      lst.remove("心理")

      ```
    
    + pop 通过索引删除，默认删除最后一个元素，并返回被删除的元素

      lst = ["大圣","海","心理","一帆","靓仔","石头"]

      print(lst.pop(1))

    + clear 清空，删除全部元素，只剩空列表
    
      ```
      lst = ["大圣","海","心理","一帆","靓仔","石头"]
      lst.clear()
      print(lst)
      ```
    
    + del 关键字删除，可删除整个列表（变量），或按索引进行删除
    
      ```
      lst = ["大圣","海","心理","一帆","靓仔","石头"]
      del lst  #删除整个列表（变量）
      del lst[0]  #通过索引进行删除
      print(lst)
      ```

      ```
    
    + 通过索引
  
  + 改：修改列表的元素（改变内存地址及地址指向）
  
    + 通过索引直接修改
    
      ```
      lst = ["大圣","海","心理","一帆","靓仔","石头"]
      lst[2] = "秀色可餐"

    + 切片修改（实质是将可迭代内容迭代覆盖切片内容）
    
      注意：使用切片时，获取的内容就是切片的原数据本身
    
      + 切片获取的内容空间是连续的时候，修改时内容可以多可以少
    
        ```
        lst = ["大圣","海","心理","一帆","靓仔","石头"]
        a = 10,20,30  #自动转换为元组
        print(a)
        lst[1:4] = 10,20,30,40,50
        ```
    
      + 切片获取的内容空间是不连续的时候，修改时内容必须一一对应
    
        ```

        lst[1:5:2] = ["alex","wusir"]
        print(lst)
        lst[1:5:2] = ["alex"]  #不会报错，但会因为缺少补充元素无法显示列表结果
        print(lst)
        ```

## 元组

Python中的数据类型之一（tuple）

+ 定义

  有序，可迭代，不可变的数据类型

  理解为：元组就是一个不可变的列表（这里的不可变指的是内存地址不变）

  ```
  tu = (1,2,3,"aaa")
  ```

+ 支持索引

  ```
  tu = (1,2,3,"aaa")
  print(tu[0])
  ```

+ 支持切片

  ```
  tu = (1,2,3,"aaa")
  print(tu[1:3])
  ```

+ 常用方法

  + 统计元素个数

    ```
    tu = (1,2,3,"aaa")
    print(tu.count("a"))
    ```

  + 通过元素的名称获取元素的索引

    ```
    print(tu.index("a")) #报错
    print(tu.index(3))
    ```

+ 应用场景

  + 为了防止误操作时修改数据，元组中存放一些重要数据

    ```
    tu = (1,2,3,4)
    tu[0] = 20 #报错
    ```

  + 配置文件中存储数据（减少命名）

+ 面试题

  ```
  a = (10)  #当小括号中没有出现逗号时，数据类型就是括号中数据类型本身
  a = ("aaa")  #当小括号中没有出现逗号时，数据类型就是括号中数据类型本身
  a = (10,) #元组
  a = () #元组
  a = (1,2,3) #元组
  print(type(a))
  ```

## range

范围：通过range达到循环数字

+ 可迭代对象

+ python2,3区别

  ```
  a = range(0,5)
  print(a)  #打印range本身 python2 [0,1,2,3,4]   python3 range(0,5)
  ```

+ 对比：和切片相似

  range(起始位置,终止位置,步长)    顾头不顾尾

  ```
  lst = list(range(0,5)) #直接打印列表
  print(lst)
  ```

  ```
  for i in range(0,51,2): #偶数
      print(i)
  for i in range(1,51,2): #奇数
      print(i)
  ```

  一般情况可以只写终止位置，使用步长时要添加起始位置

  ```
  for i in range(10,2,-1):    
  	print(i)
  	
  for i in range(100,-1,-1):    
  	print(i)
  ```
